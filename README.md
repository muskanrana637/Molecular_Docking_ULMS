🧬 Multi-Ligand Docking and Molecular Dynamics using Linux

A Linux-based workflow for performing multi-ligand molecular docking and molecular dynamics (MD) simulations for in-silico drug discovery.

This repository provides command-line pipelines, scripts, and step-by-step procedures to automate docking and simulation tasks efficiently.

📌 Project Description

Computational drug discovery relies heavily on:

Molecular Docking → predicts ligand binding affinity
Molecular Dynamics (MD) → studies stability of protein–ligand complexes

This project demonstrates how to perform both using Linux CLI tools, enabling:

High-throughput ligand screening
Automation using Bash scripting
Reproducible research workflows

Such approaches are widely used in virtual screening pipelines and drug discovery research .

🎯 Features
🧪 Multi-ligand docking automation
⚙️ Linux command-line workflow
🔁 Batch processing using shell scripts
🧬 Molecular dynamics simulation pipeline
📂 Organized structure for reproducibility
🧰 Tools Used

This project uses widely adopted open-source tools:

AutoDock Vina – docking engine
Open Babel – file conversion
GROMACS – molecular dynamics simulation
MGLTools – receptor/ligand preparation
Bash scripting – workflow automation

These tools are commonly used for large-scale docking and simulation workflows in computational chemistry .

💻 System Requirements
Linux OS (Ubuntu recommended)
Python (3.x)
Installed tools:
AutoDock Vina
Open Babel
GROMACS
MGLTools
